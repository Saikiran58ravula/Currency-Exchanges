# Currency Exchange Converter

A simple web app to convert amounts between different currencies. Built using HTML, CSS, and JavaScript as a mini project to practice working with APIs and DOM manipulation.

## Features

- Convert between major world currencies
- Real-time exchange rates
- Simple and clean interface
- Works well on both mobile and desktop

## Tech Used

- HTML
- CSS
- JavaScript

## Project Files

- index.html – main structure of the page
- style.css – styling
- script.js – handles conversion logic
- codes.js – list of currency codes used in the dropdowns

## How to Run

1. Clone or download this repository
2. Open index.html in your browser
3. Select the currencies and enter an amount to convert

## About

This project was built to understand how to fetch and use live data from an API in a frontend project. Still a work in progress — planning to improve the UI and add more features later.
